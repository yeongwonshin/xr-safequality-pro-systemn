# Customer Onboarding

## Week 0: Discovery

- 현장/작업/위험/품질 문제 정의
- 대상 SOP 수집
- 객체/상태 taxonomy 작성
- 개인정보/보안 요구사항 확인
- 성공 KPI 합의

## Week 1~2: Configuration

- 테넌트/사용자/권한 설정
- SOP DSL 초안 생성
- 현장 zone/asset 등록
- 기본 대시보드 구성
- 데이터 수집 계획 수립

## Week 3~6: Model & Procedure Setup

- 현장 이미지 수집
- 라벨링/검수
- 모델 fine-tuning
- SOP approval workflow
- Edge gateway 설치 옵션 검토

## Week 7~10: Pilot

- 제한된 작업자/절차로 운영
- false alarm/누락/UX 이슈 수집
- threshold 조정
- 관리자 리포트 리뷰
- ROI 중간 점검

## Week 11~12: Expansion Decision

- KPI 달성 여부 확인
- 보안/감사 자료 패키징
- 추가 SOP/현장 rollout 계획
- 계약/가격 확정

## 고객에게 요구할 자료

- SOP/작업표준서
- 위험성 평가 자료
- 품질 검사 기준
- 설비/부품/시약 목록
- 현장 레이아웃/zone 정보
- 개인정보/보안 정책
- 기존 MES/QMS/LIMS API 문서
