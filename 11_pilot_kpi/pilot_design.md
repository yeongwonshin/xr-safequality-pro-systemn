# Pilot Design

## Pilot Objective

XR SafeQuality Pro가 실제 현장에서 안전·품질 절차 누락을 줄이고, 작업 기록/감사 대응 비용을 절감하며, 현장 사용성을 확보하는지 검증한다.

## Pilot Scope Example

| 항목 | 설정 |
|---|---|
| 기간 | 8~12주 |
| 현장 | 연구실 1곳 또는 공정 셀 1곳 |
| 사용자 | 10~30명 |
| SOP | 3개 |
| 객체 클래스 | 10~30개 |
| 디바이스 | 스마트폰/태블릿, 필요 시 XR 2대 |
| 대시보드 | EHS/QA/Supervisor용 |

## Baseline Data

도입 전 최소 2~4주 동안 다음을 측정한다.

- 절차 누락 건수
- near-miss/incident 건수
- 품질 불량/재작업 건수
- 검사/리포트 작성 시간
- 작업자별 숙련도/오류 패턴

## Success Criteria

- critical step 누락 차단 사례 확인
- 작업자 NPS 또는 만족도 기준 이상
- false alarm이 운영 가능한 수준
- 관리자 리포트 작성 시간 절감
- 고객이 유료 확장할 ROI 근거 확보

## Pilot Deliverables

- SOP DSL 3종
- 고객 현장 모델 v1
- 대시보드와 월간 리포트
- false positive/negative 분석
- ROI 분석
- expansion roadmap
