# KPI Framework

## Safety KPIs

| KPI | 산식 | 목표 예시 |
|---|---|---:|
| PPE Compliance | PPE pass / PPE required sessions | 95%+ |
| Critical Block Detection | critical fail detected / actual critical fail | 95%+ |
| Near-miss Capture | reported near-miss after / before | 2x |
| Repeat Violation Reduction | 반복 위반 after / before | -30% |

## Quality KPIs

| KPI | 산식 | 목표 예시 |
|---|---|---:|
| First Pass Yield | first-pass sessions / total sessions | +10~20% |
| Rework Reduction | rework after / before | -20% |
| Missing Part Detection | detected missing part / actual missing part | 95%+ |
| Inspection Time | 평균 검사 시간 | -10~30% 또는 유지 |

## Adoption KPIs

| KPI | 목표 |
|---|---:|
| Weekly Active Workers | 대상자 80% 이상 |
| Session Completion Rate | 90% 이상 |
| Retake Rate | 초기 20% 이하, 개선 후 10% 이하 |
| Supervisor Override Rate | critical 정책에 따라 관리 |
| User Satisfaction | 4/5 이상 |

## AI KPIs

- class-wise F1
- critical recall
- false alarm rate
- confidence drift
- human disagreement rate
- hard negative performance
