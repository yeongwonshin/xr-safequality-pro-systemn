# Analytics Metrics

## Safety Metrics

| Metric | 정의 |
|---|---|
| PPE Compliance Rate | PPE 검증 통과 세션 / PPE 필요 세션 |
| Restricted Zone Violations | 금지구역 진입 이벤트 수 |
| Critical Blocks | critical step에서 차단된 건수 |
| Near-miss Rate | 작업 100건당 near-miss |
| Override Rate | 승인자 예외 승인 / 전체 critical step |
| Repeat Violation Rate | 동일 작업자/현장/단계 반복 위반 |

## Quality Metrics

| Metric | 정의 |
|---|---|
| First Pass Yield | 최초 검사 통과율 |
| Rework Rate | 재작업 필요 세션 / 전체 세션 |
| Missing Part Rate | 부품/나사 누락 이벤트 비율 |
| Step Defect Density | 단계별 불량 밀도 |
| Inspection Cycle Time | 검사 시작~완료 시간 |
| False Alarm Rate | 관리자/작업자에 의해 false positive로 분류된 경고 |

## AI Metrics

| Metric | 정의 |
|---|---|
| Critical Recall | critical 객체/상태 검출 recall |
| Confidence Drift | 현장/시간별 confidence 분포 변화 |
| Unknown Object Rate | taxonomy에 없는 객체 비율 |
| Retake Rate | 재촬영 요청 비율 |
| Human Override Disagreement | 사람이 AI 판단을 뒤집은 비율 |

## Business Metrics

- 작업 세션 수
- 현장 수
- SOP 수
