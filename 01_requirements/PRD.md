# PRD: XR 안전 점검·품질검사 시스템

## 1. 제품 목표

XR SafeQuality Pro는 고위험·고정밀 작업에서 작업자가 실제로 안전·품질 절차를 준수했는지 실시간 검증하고, 누락·오류·위험행동을 즉시 차단하며, 사후 감사 가능한 증적을 자동 생성한다.

## 2. 핵심 사용자 스토리

### 작업자
- 나는 작업 시작 전 필요한 PPE와 도구가 맞는지 카메라로 자동 확인받고 싶다.
- 나는 다음 단계가 무엇인지 XR 화면에서 보고, 완료 조건을 실시간으로 확인받고 싶다.
- 나는 잘못된 부품이나 시약을 집었을 때 즉시 경고받고 싶다.

### 현장 리더
- 나는 작업자가 SOP를 지켰는지 증적과 함께 확인하고 싶다.
- 나는 반복적으로 실패하는 단계와 원인을 보고 싶다.
- 나는 위험 작업에서 예외 승인을 누가 했는지 추적하고 싶다.

### 관리자/EHS/QA
- 나는 현장별 safety/quality risk score를 보고 우선순위를 정하고 싶다.
- 나는 감사 대응을 위해 작업 이력, 사진, 모델 판단 근거, 승인 로그를 내보내고 싶다.
- 나는 SOP 개정 시 어떤 작업에 영향을 주는지 알고 싶다.

## 3. 범위

### In Scope
- SOP 기반 절차 실행
- 카메라 기반 객체/상태/라벨/PPE 검출
- 순서 오류·누락·금지구역·보호구 미착용 감지
- 관리자 대시보드
- 감사 로그와 리포트
- 모델 성능/신뢰도 관리
- 현장별 정책 설정

### Out of Scope for v1
- 완전 자율 로봇 제어
- 모든 산업 표준 자동 인증
- 법적 책임 자동 판정
- 고위험 설비의 직접 제어권 획득

## 4. 성공 지표

| 지표 | 목표 |
|---|---:|
| 절차 누락 감소율 | 50% 이상 |
| near-miss 기록률 | 2배 이상 증가 |
| 품질 재작업 감소율 | 20% 이상 |
| 작업 리포트 작성 시간 | 70% 절감 |
| 작업자 단계별 평균 지연 | 10초 이하 |
| 객체 검출 중요 클래스 F1 | 0.90 이상 |
| critical false negative | 현장별 SLA 이하 |

## 5. 제품 원칙

1. **Safety first:** 모델이 확신하지 못할 때는 안전한 쪽으로 fail-safe 처리
2. **Human-in-the-loop:** 고위험 예외는 승인자 확인 필요
3. **Auditability:** 모든 판단은 시간, 모델 버전, 입력 증적, 정책 버전과 연결
4. **Privacy by design:** 얼굴/개인정보 최소 수집, 마스킹, 보존기간 정책
5. **Edge resilience:** 네트워크 장애 시에도 핵심 체크는 지속
