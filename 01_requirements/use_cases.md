# Use Cases

## UC-001: 실험실 시약 라벨 확인

**Actor:** 연구원  
**Precondition:** 작업자는 실험 SOP를 선택하고 카메라 접근 권한을 허용했다.  
**Flow:**
1. 시스템이 필요한 시약 목록과 위험 등급을 표시한다.
2. 작업자가 시약병 라벨을 카메라에 비춘다.
3. OCR/객체 인식이 라벨, 농도, 유효기간, GHS pictogram을 확인한다.
4. 불일치 시 경고하고 다음 단계 진입을 차단한다.
5. 일치 시 증적 이미지를 저장하고 다음 단계로 이동한다.

**Success Criteria:** 잘못된 시약 또는 농도 불일치가 critical false negative 없이 차단된다.

## UC-002: 보호구 미착용 차단

1. 작업자가 고위험 구역 입장 전 체크를 시작한다.
2. 카메라가 보안경, 장갑, 실험복, 마스크를 검출한다.
3. 누락 PPE가 있으면 XR 오버레이로 해당 항목을 표시한다.
4. 작업자가 보완 후 재검증한다.
5. 성공 시 입장 승인 로그가 생성된다.

## UC-003: 조립 공정 나사 누락 검사

1. 작업자는 제품/공정 ID를 스캔한다.
2. 시스템은 해당 공정의 나사 위치 템플릿을 로드한다.
3. 카메라 영상에서 체결 위치와 나사 존재 여부를 비교한다.
4. 누락 또는 잘못된 위치가 있으면 오버레이로 표시한다.
5. 재작업 후 합격 리포트를 생성한다.

## UC-004: 설비 점검 순서 강제

1. 작업자는 설비 PM 작업을 시작한다.
2. 시스템은 Lockout/Tagout, 전원 상태, 밸브 위치, 커버 개방, 센서 점검 순서를 강제한다.
3. 이전 단계 증적이 없으면 다음 단계로 진행할 수 없다.
4. 예외 발생 시 승인자를 호출한다.
5. 작업 종료 후 설비 ID별 PM 이력이 저장된다.

## UC-005: 금지구역 진입 감지

1. 작업자가 위험구역 근처에서 작업한다.
2. XR/모바일 카메라 또는 고정 카메라가 구역 경계를 인식한다.
3. 사람이 경계 안으로 진입하면 즉시 경고한다.
4. 경고 후 3초 이상 체류 시 관리자에게 알림을 보낸다.
5. 이벤트는 near-miss로 기록된다.
