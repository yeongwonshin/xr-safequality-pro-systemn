# Target Segments & Buyer Persona

## 1. 연구실/실험실

### 구매자
- EHS 담당자
- 연구소장/랩 매니저
- 대학 산학협력단/안전관리팀

### 사용자
- 연구원, 대학원생, 실험 조교, 장비 담당자

### 긴급 Pain Point
- 신규 인력의 SOP 미준수
- 유해화학물질 라벨 확인 누락
- 보호구 착용의 사각지대
- 폐기물 분류 오류

### 도입 메시지
“실험 전·중·후 절차를 카메라 기반으로 검증하고, 사고 발생 전 누락을 잡아 감사 가능한 실험 안전 기록을 자동 생성합니다.”

## 2. 배터리/반도체 제조

### 구매자
- 품질보증(QA) 팀장
- 생산기술/설비보전 팀장
- 안전환경팀
- 공장장

### 사용자
- 조립 작업자, 설비 점검자, 검사자, 교대조 리더

### 긴급 Pain Point
- 부품 방향/사양 오류
- 나사 체결 누락
- 설비 PM 절차 누락
- 작업자 숙련도 편차

### 도입 메시지
“작업자의 손 앞에서 누락·오조립·순서 오류를 즉시 검출하고, 라인별 불량 원인을 데이터화합니다.”

## 3. 화학/바이오 시설

### 구매자
- EHS, Compliance, QA, Validation 담당자

### 사용자
- 실험자, 제조 오퍼레이터, QC 분석자

### 긴급 Pain Point
- 라벨/용기 혼동
- PPE/후드/폐기 절차 누락
- 감사 대응 증적 부족

### 도입 메시지
“고위험 작업 절차를 AI가 현장에서 검증하고, 예외·승인·증적을 감사 로그로 남깁니다.”
