# Product Roadmap

## Phase 0: Foundation
- SOP DSL 확정
- 모바일 웹 카메라 기반 점검 플로우
- 기본 객체 검출: PPE, 라벨, 용기, 나사, 밸브
- 관리자 대시보드 v0
- 감사 로그 스키마

## Phase 1: Professional SaaS
- 멀티테넌트 조직/현장/작업자 권한
- 절차 버전관리와 승인 워크플로우
- Edge inference gateway
- 이벤트 기반 알림
- 모델 성능 모니터링
- PDF/CSV 감사 리포트

## Phase 2: Industrial Integration
- MES/QMS/LIMS/EHS API 연동
- SSO/SAML/OIDC
- 현장별 모델 fine-tuning 워크플로우
- 금지구역/위험구역 맵핑
- 설비 PM 템플릿
- QR/RFID/Barcode 연계

## Phase 3: XR Glasses Premium
- HMD UI, 음성 명령, 핸즈프리 모드
- 3D 위치 앵커와 공간 체크포인트
- 작업자 시야 기반 step-by-step guidance
- 오프라인 동기화
- 디바이스 관리 정책

## Phase 4: Predictive Safety & Quality
- 반복 오류 기반 선제 코칭
- 공정별 위험 점수 예측
- 모델 active learning 자동화
- 디지털 트윈/라인 밸런싱 연계
- 보험/감사/규정 리포팅 패키지
