# ISO 9001 Alignment Mapping

> 목적: 품질관리 시스템 관점에서 XR SafeQuality Pro의 기여 영역을 정의합니다. 인증 대체물이 아니라 QMS 운영을 보조하는 디지털 증적 시스템입니다.

## Alignment Areas

| ISO 9001 관점 | 시스템 기능 | 증적 |
|---|---|---|
| 운영 계획 및 통제 | SOP 기반 품질 gate | 작업 세션 이력 |
| 제품/서비스 요구사항 | 공정별 검사 기준 등록 | 검사 조건/허용 기준 |
| 생산 및 서비스 제공 | 실시간 조립/점검 검증 | 부품/나사/상태 증적 |
| 식별 및 추적성 | 제품/설비/작업자/시간 연결 | session traceability |
| 부적합 출력 관리 | fail/rework 이벤트 | 불량 리포트, 재검사 이력 |
| 성과 평가 | FPY, rework, defect trend | 품질 KPI 대시보드 |
| 개선 | 반복 불량 원인 분석 | CAPA/QMS 연동 데이터 |

## QMS Integration

- QMS에 nonconformance ticket 생성
- 검사 결과를 제품 lot/batch/serial과 연결
- SOP 변경관리와 품질 문서관리 시스템 연동
- CAPA 상태와 XR 검사 결과 cross-reference
