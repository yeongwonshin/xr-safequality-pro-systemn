# OSHA Laboratory Safety / Chemical Hygiene Plan Mapping

> 목적: 실험실 고객을 위한 Chemical Hygiene Plan(CHP) 및 PPE/작업 절차 관리 보조 기능을 정리합니다. 지역별 법규와 기관 정책은 별도 검토해야 합니다.

## System Support

| 실험실 안전 요소 | 시스템 기능 |
|---|---|
| Chemical Hygiene Plan 문서화 | SOP/절차 버전관리와 승인 워크플로우 |
| PPE 요구사항 | 단계별 PPE 검출과 미착용 차단 |
| 유해화학물질 취급 절차 | 시약 라벨/OCR/GHS pictogram 확인 |
| 장비/후드 사용 | 후드 sash, 장비 상태, 폐기 용기 확인 |
| 폐기 절차 | 폐기물 라벨/용기/분류 확인 |
| 교육과 기록 | 작업자별 세션/오류/재시도 이력 |
| 사고/near-miss | incident 이벤트와 관리자 알림 |

## Lab SOP Template

- 실험 전: PPE, 시약, SDS/위험표지, 후드, 폐기용기
- 실험 중: 장비 상태, 구역 진입, 라벨 유지, spill response
- 실험 후: 폐기 분류, 장비 off, 후드 정리, 증적 리포트

## Evidence Fields

- 작업자/승인자
- SOP/version
- 시약/장비/폐기 카테고리
- PPE 검증 결과
- 이미지 증적 hash
- 예외 승인 사유
