# XR/Mobile UX Flow

## 작업자 플로우

1. 로그인 또는 SSO
2. 현장/설비/작업 선택 또는 QR 스캔
3. SOP 버전 확인
4. 위험 공지와 필수 PPE 표시
5. 단계별 XR 오버레이 안내
6. 카메라 기반 자동 검증
7. 경고/재촬영/승인 요청
8. 작업 완료 리포트 확인

## 화면 구조

### 작업 시작 화면
- 작업명, 설비/구역, 예상 소요시간
- 위험 등급
- 필수 PPE
- “시작 전 확인” 버튼

### 실시간 검사 화면
- 중앙: 카메라 뷰
- 상단: 현재 단계/전체 단계 진행률
- 하단: 다음 액션, 음성 안내
- 오버레이: 누락 객체 위치, 금지구역 경계, 부품 방향
- 경고: 색상/진동/음성/텍스트 조합

### 예외 승인 화면
- 실패 조건
- 증적 이미지
- 작업자 사유 입력
- 승인자 호출
- 승인자 PIN/SSO 확인

### 완료 화면
- 전체 pass/fail 요약
- 재시도 횟수
- 예외 승인 목록
- 리포트 링크

## Alert Design

| 유형 | UX |
|---|---|
| 정보 | 작은 안내 카드 |
| 주의 | 노란 오버레이, 재확인 요청 |
| 위험 | 빨간 오버레이, 음성/진동, 진행 차단 |
| 관리자 호출 | 승인 요청 modal |

## 작업자 부담 최소화

- 가능한 자동 검증
- 수동 입력 최소화
- 장갑 착용 시 큰 버튼과 음성 입력
- 반복되는 정상 단계는 자동 통과하되 증적 저장
- 네트워크 지연 시 로컬 캐시로 계속 진행
