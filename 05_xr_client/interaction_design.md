# Interaction Design

## Input Methods

| 환경 | 입력 방식 |
|---|---|
| 스마트폰 | 터치, 카메라, QR/바코드, 음성 |
| 태블릿 | 터치, 외부 스캐너, 카메라 |
| XR 글래스 | 시선, 제스처, 음성, 물리 버튼 |
| 클린룸 | 음성, 큰 버튼, 무접촉 제스처 |

## XR Overlay Rules

- 검출 객체는 bounding outline과 label로 표시한다.
- 누락 위치는 ghost marker로 표시한다.
- 금지구역은 반투명 경계와 진입 방향 경고로 표시한다.
- 작업 단계별로 필요한 정보만 보여준다.
- Critical warning은 주변 시야 방해를 최소화하되 인지 가능해야 한다.

## Voice Commands

- “다음 단계”
- “다시 검사”
- “승인 요청”
- “증적 저장”
- “작업 중지”
- “도움말”

## Accessibility

- 색상만으로 상태를 구분하지 않는다.
- 텍스트, 아이콘, 진동/음성을 병행한다.
- 소음 환경에서는 진동/시각 알림을 강화한다.
- 장갑 착용을 고려해 최소 터치 타겟을 크게 유지한다.
